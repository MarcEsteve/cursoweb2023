Ejercicio:
Busca la media de precios de Deportes y muestra los precios que sean mayores que esa media

Primero consulta "padre":

SELECT SECCIÓN, NOMBREARTÍCULO, PRECIO 
FROM productos 
WHERE PRECIO > (???MEDIA*);

Consulta "hijo" para encontrar la media de ferretería:

SELECT AVG(PRECIO) 
FROM productos
WHERE SECCIÓN='DEPORTES';

SELECT SECCIÓN, NOMBREARTÍCULO, PRECIO 
FROM productos 
WHERE PRECIO > (SELECT AVG(PRECIO) 
FROM productos
WHERE SECCIÓN='DEPORTES');
